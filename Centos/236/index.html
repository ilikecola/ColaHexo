CentOS下查看、更新、删除yum安装包
2017-02-14 • 分类: Centos • 标签: Centos, Linux

查看CentOS下安装了哪些yum软件包，查看指定包是否有安装，查看已安装包详细说明信息、更新软件包、删除软件包等常用操作。

查看搜索软件包

  1. 查找指定软件包
# shell sentence
$ yum search softwate_package_name

# for instance(调制解调器通讯程序：使用rz和sz实现终端模拟软件的上传和下载)
$ yum search lrzsz

  1. 列出所有可安装的软件包
# shell sentence
$ yum list

  1. 列出所有可更新的软件包
# shell sentence
$ yum list updates

  1. 列出所有已安装的软件包
# shell sentence
$ yum list installed

  1. 列出所有已安装但不在Yum Repository软件包资源库内的软件包
# shell sentence
$ yum list extras

  1. 列出指定的软件包安装信息
# shell sentence
$ yum list software_package_name

# for instance(调制解调器通讯程序：使用rz和sz实现终端模拟软件的上传和下载)
$ yum list lrzsz

  1. 查看软件包说明信息
# shell sentence
$ yum info software_package_name

# for instance(调制解调器通讯程序：使用rz和sz实现终端模拟软件的上传和下载)
$ yum info lrzsz

  1. 查看所有软件包说明信息
# shell sentence
$ yum info

更新软件包

# shell sentence
$ yum update software_package_name

# for instance(调制解调器通讯程序：使用rz和sz实现终端模拟软件的上传和下载)
$ yum update lrzsz

更新系统

# shell sentance
$ yum update

卸载软件包

# shell sentence
$ yum remove software_package_name

# for instance(调制解调器通讯程序：使用rz和sz实现终端模拟软件的上传和下载)
$ yum remove lrzsz
