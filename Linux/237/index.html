Linux Recovery模式下root用户只读的对策
2017-02-18 • 分类: Linux • 标签: Linux, Ubuntu

某次在Recovery中修改文件时，提示【该文件为只读模式】,明明是root权限怎么会是只读？

原因:
因为Recovery在挂载时写入了只读，因此即使root权限也无法修改

对策:

mount -o remount,rw /
